소개

소프트웨어 프로그래머 프리랜서

최근 프로그래밍 트랜드를 보면서 다양한 라이브러리를 활용하면 하나의 소프트웨어 시스템을 여러 사람이 아닌 혼자서 구축할 수 있다는 생각이 들었습니다. 그래서 저는 2013년부터 여러가지 Swift, Kotlin, Node.js 등등의 프로그래밍 언어들을 리서치하고 실무에 참여할 수 있을 때마다 참여해서 다양한 시스템 환경에 대한 노하우를 익혀왔습니다.

제가 제공할 수 있는 개발 서비스는 고객용 안드로이드, 아아폰 앱을 만들어 드리고, 서비스를 연결하는 데이터베이스 서버를 만들어 드리고, 이들을 관리하는 관리자용 앱 서비스를 만들어드리는 일입니다. 이정도 규모의 개발을 해야 하는 경우 보통 안드로이드 개발자, 아이폰 개발자, 백앤드 개발자, 프론트앤드 개발자, 프로젝트 관리자들로 보통 5명 정도의 사람이 필요합니다. 저는 이정도 규모의 일을 혼자 개발할 수 있습니다.

저에게 메일을 주시면 적절한 가격으로 견적을 내서 보내 드리겠습니다.
